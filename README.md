# 简介：带上你的蚂蚁大军，扎根每一座空岛吧！
点击两座空岛，进行蚂蚁的转移。每座岛有自己的编号、繁殖速度和初始数量（有的岛蚂蚁繁殖速度为0（悲））
注意，一座空岛上蚂蚁数量为0时，不会产生新的蚂蚁。
当你的蚂蚁征服了所有空岛后，就可以去嚯嚯下片空岛群了（bushi
